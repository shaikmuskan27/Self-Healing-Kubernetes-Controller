Autonomous Self-Healing Kubernetes Controller

A custom Python Operator that detects Pod failures, analyzes logs using an intelligent diagnostic engine, and applies automated remediation via GitOps pull requests.

Explore Architecture Source Code

The Workflow

From failure detection to fully automated GitOps remediation.

01

Detection

Controller listens to the K8s API for Pods entering CrashLoopBackOff or Error states.

02

Diagnostics

Fetches and parses the last 50 lines of container logs to identify the root cause (e.g., OOMKilled).

03

GitOps Remediation

Leverages GitHub API to dynamically patch deployment limits and open an Auto-Fix Pull Request.

04

Notification

Fires a rich, interactive Slack/Teams webhook detailing the incident and linking the PR.
